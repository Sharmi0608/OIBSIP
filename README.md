Project Overview
Lumira is a Python-based AI Voice Assistant developed as part of the Oasis Infobyte Python Programming Internship.

The assistant can recognize voice commands and perform various tasks such as:

Opening websites
Playing songs on YouTube
Providing weather information
Searching Google
Opening applications
Taking screenshots
Setting reminders
Providing battery and CPU information
Taking notes
Telling jokes
Answering basic questions

Installation
Install the required libraries:

pip install -r requirements.txt
Running the Project
Open the terminal inside the Lumira folder and run:

py -3.12 main.py
