## Internship Project

### Task 1: AI Voice Assistant (Lumira)

Lumira is a Python-based AI Voice Assistant capable of understanding voice commands and performing various desktop and web automation tasks.

### Features

* Speech Recognition
* Text-to-Speech Responses
* Weather Information
* YouTube Music Playback
* Google Search
* System Controls
* Screenshot Capture
* Reminder System
* Battery Monitoring
* Notes Taking
* Wikipedia Search
* Desktop Application Launching

---

## Technologies Used

* Python 3.12
* SpeechRecognition
* pyttsx3
* pywhatkit
* Wikipedia API
* Requests
* psutil
* pyautogui
* spaCy
